# Curl Testing Commands

## Basic Checks

# Check if site is up (headers only)
curl -I https://fusion.paperfrogs.dev/

# Get full page content
curl https://fusion.paperfrogs.dev/

# Follow redirects (-L)
curl -L https://fusion.paperfrogs.dev/

# See response time
curl -w "Time: %{time_total}s\n" -o /dev/null -s https://fusion.paperfrogs.dev/

# Check specific endpoint
curl https://fusion.paperfrogs.dev/status.html

# See all headers + body (verbose)
curl -v https://fusion.paperfrogs.dev/


## API Testing

# Test API endpoint with POST
curl -X POST https://fusion.paperfrogs.dev/.netlify/functions/send-contact \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","message":"hello"}'

# Test waitlist signup
curl -X POST https://fusion.paperfrogs.dev/.netlify/functions/send-early-access \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com"}'


## Rate Limiting

# Check if rate limiting works
curl -I https://fusion.paperfrogs.dev/ | grep -i ratelimit

# Test multiple times quickly (check rate limits)
for i in {1..5}; do curl -s -o /dev/null -w "%{http_code}\n" https://fusion.paperfrogs.dev/; done


## Debugging

# Check HTTP status code only
curl -s -o /dev/null -w "%{http_code}" https://fusion.paperfrogs.dev/

# Check SSL certificate info
curl -vI https://fusion.paperfrogs.dev/ 2>&1 | grep -i "SSL\|certificate"

# Download and save response
curl -o response.html https://fusion.paperfrogs.dev/

# Check if assets load
curl -I https://fusion.paperfrogs.dev/assets/index-DtaF0534.js


## Quick Reference

# -I          Headers only
# -L          Follow redirects
# -s          Silent (no progress bar)
# -v          Verbose (show everything)
# -X POST     POST request
# -H          Add header
# -d          Send data/body
# -o file     Save output to file
# -o /dev/null   Discard output
# -w          Custom output format
